## Factorial maker
```
function factorial (n) {
	 
	 	if (n<=1)
	 	   return 1    //non-recursive exit
	 	else
	 	   return n * factorial(n-1);
	 
	 } // end function factorial 
```
## Fibonacci 
```
function Fibonacci(n){
     if (n<=1)
       return n;
	
	else
	   return Fibonacci(n-1) + Fibonacci(n-2);
	
} // end factorial
```
