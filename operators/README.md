## Switch

```
var score =   Math.floor(Math.random() * 10) + 1;
	
	switch (score) {
	
		case 10, 9: grade="A";
		            break;
		case     8: grade="B";
		            break;
		case     7: grade="C";
		            break;
		default: grade = "'other'";
	
	
	
	
	
	} // end switch (score)
```
## Do-While
```
do {
	      var score = Math.floor (Math.random() * 11 + 0);
	    
	 
	      alert("Score was " + score);
	       // Math.random gives you a num such that 0<=num<1
	    } while (score < 6)  /*has to choose a score 6 or greater!  */
	    
	    
```
